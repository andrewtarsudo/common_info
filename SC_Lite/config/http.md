В файле настраиваются параметры соединений по протоколу HTTP.

Список разделов:
* **[Server](#server)**
  * **[SSL](#sslserver)**
  * **[Authorization](#authorizationserver)**
* **[Client](#client)**
  * **[SSL](#sslclient)**
  * **[Authorization](#authorizationclient)**
* **[AdditionalDir](#additionaldir)**

### <a name="sslversion">SSL Version </a>
Возможные значения:
* 1 – TLSv1;
* 2 – SSLv2;
* 3 – SSLv3;
* 4 - все версии.

### <a name="httpversion">HTTP Version</a>
Возможные значения:
* 1 – HTTP/1.0, HTTP/1.1;
* 2 – HTTP/2.

### <a name="type">Type</a>
Возможные значения:
* 0 - любой;
* 1 - Basic;
* 2 - Digest.
